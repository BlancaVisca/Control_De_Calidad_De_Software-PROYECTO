# RENOVABLE

RENOVABLE es un juego educativo de cartas digitales enfocado en la separación de residuos como eje principal del aprendizaje ambiental. El objetivo es enseñar de forma interactiva la correcta clasificación de desechos mediante una dinámica estratégica por turnos entre el jugador y un oponente controlado por el sistema.

El juego se centra en cuatro tipos de reciclaje: materiales reciclables, inorgánico no reciclable, metales y orgánico. Cada tipo incluye cartas numeradas del 1 al 9 y colores independientes (rojo, azul, amarillo y verde), además de cartas comodín con efectos especiales que modifican las reglas de la partida y refuerzan la toma de decisiones.

Desde el punto de vista técnico, RENOVABLE está desarrollado como una aplicación de software interactiva para computadoras personales. Funciona sobre un sistema operativo compatible con navegadores modernos y utiliza teclado y ratón como dispositivos de entrada, con la pantalla como principal medio de salida. No requiere hardware especializado ni periféricos adicionales.

A nivel de software, el proyecto integra lógica de juego por turnos, validación automática de reglas, gestión de mazos, control de vidas y un sistema de preguntas que permite modificar el flujo del juego. El oponente es gestionado por algoritmos que simulan decisiones estratégicas, garantizando una experiencia fluida y completamente autónoma.
